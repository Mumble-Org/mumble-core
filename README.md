## Technologies

- Node.js
- Express.js
- Typescript
- Ubuntu 14.0

## Installation

### Clone repository

`git clone https://github.com/Mumble-Org/mumble-core`

### Install dependencies

`cd mumble-core`
`npm install`

## Commands

- Compile typescript
  `npm run build`
- Start Server
  `npm run start` or `npm run dev` to watch
- Run Linter(eslint) and fix
  `npm run lint`
- Run Tests
  `npm run test`
